# ecommerce-sales-dashboard-powerbi
## Overview
An interactive Power BI dashboard built using a Blinkit-style e-commerce dataset to analyze sales performance and key business metrics.

## Key Insights
- Sales and revenue trends
- Top products and categories
- City-wise sales performance
- Monthly growth analysis
